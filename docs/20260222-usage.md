# 使い方

## 前提

- Rust/Cargo が利用可能
- Marp CLI が利用可能（`config.toml` の `build.marp_binary` で指定）

## インストール

```bash
cargo install slide-flow
```

## 初期化

任意の空ディレクトリで実行します。

```bash
slide-flow init
```

作成される最小構成:

```text
.
├── .marp/
│   └── themes/
│       └── .gitkeep
├── src/
│   └── .gitkeep
└── config.toml
```

## スライド追加

```bash
# 公開スライドとして追加
slide-flow add intro --secret false

# 秘密スライドとして追加（既定）
slide-flow add internal

# 下書きスライドとして追加
slide-flow add draft-topic --secret false --draft true
```

作成される構成:

```text
src/<name>/
├── images/
│   └── .gitkeep
├── slide.md
└── slide.toml
```

## インデックス更新

```bash
# 1スライドだけ更新
slide-flow index --dir src/intro

# 全スライド更新
slide-flow index
```

- `slide.toml` の `title_prefix`（既定 `# `）に一致する見出しを収集
- 同タイトルが連続する場合は `(1/N)` 形式で枝番を付与
- 目次を標準出力に出す（`--quiet` で抑制）

## 参考文献更新

```bash
slide-flow bib src/intro
```

- `slide.toml` の `[[bibliography]]` を参照
- 本文の `[#](#tag)` 形式リンクをページ付きアンカーへ更新
- 各ページの `<div class="footnote">...</div>` を再構築

## 公開ファイル再生成（pre-commit）

```bash
slide-flow pre-commit
```

- `README.md` を再生成
- `output/index.html` を再生成
- 現在のスライド設定に存在しない古い成果物を `output/` から削除

## バージョンを上げる（bump）

```bash
slide-flow version bump --dir src/intro
```

- `src/intro/v<current-version>/` に旧版を退避
- `src/intro/slide.toml` を `version + 1` に更新
- `src/intro/` 直下に新しい作業用ファイルを再初期化

## ビルド

```bash
# 単体ビルド
slide-flow build src/intro

# 複数ビルド
slide-flow build src/intro src/chapter2

# 並列度を指定（既定4）
slide-flow build src/intro src/chapter2 --concurrent 8
```

Marp スライドの場合 (`stem = secret` があればそれを優先、なければ `name`):

- HTML（最新版のみ）: `output/<stem>_v<version>/index.html`
- PDF（版付き）: `output/<stem>_v<version>.pdf`
- 画像: `src/<name>/images/*` を `output/<stem>_v<version>/images/` へコピー

Ipe スライド（`slide.ipe`）の場合:

- `src/<name>/slide.pdf` を `output/<stem>_v<version>.pdf` へコピー

## 設定ファイル

### `config.toml`（プロジェクト全体）

```toml
name = "my-slide-project"
author = "Your Name"
base_url = "https://example.com/"
output_dir = "output"

[template]
slide = ""
index = ""
suffix = ""

[build]
theme_dir = ".marp/themes"
marp_binary = "marp"
```

### `src/<name>/slide.toml`（スライド単位）

```toml
name = "intro"
version = 1
secret = "xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx" # 省略可
custom_path = ["intro-latest"]                  # 省略可
draft = false                                    # 省略可
description = "導入スライド"                     # 省略可
title_prefix = "# "                             # 省略可

[[bibliography]]
tag = "vaswani2017"
authors = "A. Vaswani et al."
title = "Attention Is All You Need"
year = 2017
venue = "NeurIPS"
url = "https://arxiv.org/abs/1706.03762"
```

## 日常運用の例

```bash
slide-flow add intro --secret false
slide-flow version bump --dir src/intro
slide-flow index --dir src/intro
slide-flow bib src/intro
slide-flow pre-commit
slide-flow build src/intro --concurrent 4
```
