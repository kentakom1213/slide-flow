# slide-flow 概要

`slide-flow` は、Marp 形式のスライド運用を補助する Rust 製 CLI です。  
スライドの作成、目次整備、公開用ページ生成、HTML/PDF ビルドを一貫して扱えます。

## 何を解決するか

- スライドごとの雛形作成を標準化する
- 公開対象（公開/下書き/秘密スライド）の管理を簡単にする
- `README.md` と `index.html` の更新を自動化する
- 複数スライドのビルドを並列化する

## 主な機能

- `init`: プロジェクト初期化
- `add`: 新規スライド追加（`secret` / `draft` の属性付与）
- `index`: 見出しにページ番号を付与し、目次を生成
- `bib`: 参照リンクと脚注の参考文献を更新
- `pre-commit`: 公開一覧の生成 + 出力キャッシュ整理
- `build`: Marp スライドを HTML/PDF に変換（Ipe は PDF コピー）

## このツールの前提

- スライド本体は `src/<slide-name>/slide.md`（または `slide.ipe`）
- スライド設定は `src/<slide-name>/slide.toml`
- プロジェクト設定はルートの `config.toml`
- ビルド時に `config.toml` の `build.marp_binary`（既定値: `marp`）を利用
